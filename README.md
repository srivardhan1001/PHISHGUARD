PhishGuard – Phishing Detection System

PhishGuard is a cybersecurity-focused web application designed to help users identify and report phishing websites. The system allows users to submit suspicious URLs and check whether a website may be unsafe, helping improve awareness and protection against online fraud.

Features

URL submission to check suspicious websites

Detection of potentially malicious or phishing links

Simple and user-friendly interface

Option to report suspicious websites

Helps users stay aware of online security threats

Tech Stack

Python

Django

HTML

CSS

JavaScript

MySQL

How It Works

Users enter or submit a suspicious URL.

The system analyzes the URL and checks for phishing indicators.

The result is displayed to inform the user whether the link appears safe or suspicious.

Users can report suspicious websites to help improve detection.

Installation

Clone the repository

git clone https://github.com/yourusername/phishguard.git

Navigate to the project folder

cd phishguard

Install dependencies

pip install -r requirements.txt

Run the development server

python manage.py runserver

Open in browser

http://127.0.0.1:8000/
